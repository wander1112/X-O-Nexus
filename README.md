## üïπÔ∏è Game Rules (Quick Summary)

- The board consists of **9 small Tic-Tac-Toe boards** forming one big board
- Winning a small board claims that position on the big board
- The next move location depends on the previous move
- First player to win the **big board** wins the game
- If a board is full and undecided, it is marked as a draw
